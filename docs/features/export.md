# 导出便条

读完一篇文档，你写下的便条往往就是思考的精华。**导出便条**让你把整页的便条挑一挑、整理成 Markdown —— 贴给 AI 当上下文，或存档备查。

> 只导出**带便条**的批注（你写了文字的那些）。纯高亮不会进入导出。

## 快速使用

在 `H1` 标题工具栏点击 **导出便条**（在「打印」旁边），进入一个全屏的两步向导：

1. **选择** —— 勾选要导出哪些便条
2. **整理** —— 生成 Markdown，复制或下载

## 第一步：选择内容

全屏列表按**所在标题分组**，列出本页每一条便条。默认全部勾选。顶栏一行排开：关闭、标题、**全选** / **全不选**、已选计数、**下一步 →**。

| 操作 | 效果 |
|------|------|
| 逐条勾选 | 精确控制单条的去留 |
| 分组勾选 | 整节一键全选 / 取消 |
| 全选 / 全不选 | 顶栏按钮，一步到位 |

每条左侧是便条文字（主），下方灰色一行是它引用的原文（次）。

## 第二步：整理为 Markdown

点击 **下一步 →**，选中的便条按文档顺序渲染成 Markdown，并载入编辑器（复用[快捷编辑](/features/edit)的同款界面，可编辑、实时预览）：

| 操作 | 效果 |
|------|------|
| **复制文本** | 复制全部内容；选中部分文字时按钮变为 **复制选中**，只复制选中段 |
| **下载 .md** | 保存为本地 Markdown 文件（文件名取自文档标题） |
| **← 返回** | 回到第一步重新挑选，已勾选状态保留 |

导出的 Markdown 直入正题 —— 没有标题、计数、分节或编号，每条便条就是「引文 + 便条」：

```markdown
"brown fox"
> 主语是动物

"Lorem ipsum"
> 占位用的拉丁文
```

> 只携带你的便条与其引用的原文，不含文档其余正文 —— 适合直接贴进外部 AI 工具，而不暴露全文。

## 单条快速复制

不想走整个向导？每条便条都能单独复制 —— 在便条卡片（或窄屏弹窗）上点 **复制** 图标，成功后图标会变成绿色对勾。

复制出的是以这一条为主体的「引用 + 便条」：

```markdown
> brown fox

主语是动物
```
