# 导出便条

读完一篇文档，你写下的便条往往就是思考的精华。**导出便条**可以把整页或某个章节范围内的便条整理成 Markdown，贴给 AI 当上下文，或存档备查。

> 只导出**带便条**的批注（你写了文字的那些）。纯高亮不会进入导出。

## 导出范围

| 入口 | 导出范围 |
|------|----------|
| `H1` 标题后的 **导出便条 (N)** | 当前页面的全部便条 |
| 聚焦 `H2`-`H6` 后的 **导出便条 (N)** | 当前章节及其子章节中的便条 |

章节范围从当前标题开始，到下一个同级或更高级标题之前结束。因此，在父章节执行导出会包含其所有子章节；在子章节执行导出只包含该子章节范围。按钮中的 `N` 会按对应范围实时计数，没有便条时按钮不可用。

## 整理为 Markdown

点击导出后，范围内的便条按文档顺序生成 Markdown，并直接载入编辑器（复用[快捷编辑](/features/edit)的同款界面，可编辑、实时预览）：

| 操作 | 效果 |
|------|------|
| **复制文本** | 复制全部内容；选中部分文字时按钮变为 **复制选中**，只复制选中段 |
| **下载 .md** | 保存为本地 Markdown 文件（文件名取自文档标题） |

导出的 Markdown 直入正题 —— 没有标题、计数、分节或编号，每条便条就是「引文 + 便条」：

```markdown
"brown fox"
> 主语是动物

"Lorem ipsum"
> 占位用的拉丁文
```

> 只携带你的便条与其引用的原文，不含文档其余正文 —— 适合直接贴进外部 AI 工具，而不暴露全文。

## 单条快速复制

只需要一条时，每条便条都能单独复制。在便条卡片（或窄屏弹窗）上点 **复制** 图标，成功后图标会变成绿色对勾。

复制出的是以这一条为主体的「引用 + 便条」：

```markdown
> brown fox

主语是动物
```
