# 注解与笔记

<div class="feature-illustration">
  <img src="/illustrations/05-annotate.svg" alt="注解与笔记" />
</div>

批注是 Markon 的核心能力 —— 让你在阅读时能随手做标记、加笔记，就像看实体书时的荧光笔和便条纸。

## 快速使用

**选中文本** → 选区附近自动弹出工具栏 → 点击选择操作：

<!-- TODO: screenshot: 选择工具栏 (/screenshots/annot-toolbar.png) -->

| 操作 | 效果 |
|------|------|
| 🟠 橙色高亮 | 批注重要内容 |
| 🟢 绿色高亮 | 批注赞同/正确的内容 |
| 🟡 黄色高亮 | 批注待确认/存疑的内容 |
| <s>删除线</s> | 批注已过时/错误的内容 |
| 📝 便条 | 加一段文字笔记，点击已批注的文本可查看/编辑 |

## 便条（Note）

点击 **📝 便条** 后会弹出输入框，写完保存即可在文本上形成可点击的批注。

### 显示方式

便条根据屏幕宽度自适应：

- **宽屏（≥ 1400px）** — 便条以卡片形式展示在右侧边栏，智能定位避开重叠
- **窄屏** — 便条以弹窗形式贴近文本显示，点击文本触发

### 管理

- **查看/编辑** — 点击已批注的文本
- **删除便条** — 在弹出的便条上点击删除按钮
- **取消高亮** — 再次选中已高亮的文本，工具栏会出现 **取消高亮** 选项

## 清除全部

**工具栏 → 更多 → 清除本页所有批注** — 一键清空当前文件的所有批注（有确认对话框）

## 存储与共享

| 模式 | 存储位置 | 同步 |
|------|---------|------|
| **个人** | 本机 SQLite | 仅管理员会话可见 |
| **共享** | 同一 SQLite 数据集 | 跨浏览器/设备实时同步 |

→ 共享模式详见 [共享批注](/advanced/shared-annotations)

## 撤销 / 重做

所有批注操作支持撤销/重做：

| 快捷键 | 功能 |
|--------|------|
| <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>Z</kbd> | 撤销 |
| <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>Z</kbd> | 重做 |
| <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>Y</kbd> | 重做（替代） |

默认栈深度 50。

## 在批注间跳转

| 快捷键 | 功能 |
|--------|------|
| <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>J</kbd> | 下一个批注 |
| <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>K</kbd> | 上一个批注 |

## 打印

打印时所有批注默认保留（高亮颜色、删除线、便条内容）。如不想打印批注，可临时清除或使用浏览器的「无样式打印」。

## 导出

读完后想把便条整理给 AI 或存档？整页便条可挑选、整理为 Markdown 复制或下载 —— 详见 [导出便条](/features/export)。
