# 键盘快捷键

在浏览器页面中按 <kbd>?</kbd> 可随时调出完整快捷键面板。本页作为文字版参考。

## 核心

| 快捷键 | 功能 |
|--------|------|
| <kbd>?</kbd> | 显示/关闭快捷键面板 |
| <kbd>Esc</kbd> | 关闭弹窗 / 取消选择 / 取消聚焦 |
| <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>Z</kbd> | 撤销 |
| <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>Shift</kbd> + <kbd>Z</kbd> | 重做 |
| <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>Y</kbd> | 重做（替代） |
| <kbd>/</kbd> | 打开全文搜索 |
| <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>\\</kbd> | 切换/聚焦目录（TOC） |

## 导航

| 快捷键 | 功能 |
|--------|------|
| <kbd>j</kbd> / <kbd>k</kbd> | 下/上一个标题 |
| <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>j</kbd> / <kbd>k</kbd> | 下/上一个标注 |
| <kbd>Space</kbd> | 向下平滑滚动 1/3 页（<kbd>Esc</kbd> 停止） |

## 章节控制

| 快捷键 | 功能 |
|--------|------|
| <kbd>o</kbd> | 切换当前章节折叠/展开 |
| <kbd>v</kbd> | 切换当前章节已读状态（需启用） |

## 编辑

| 快捷键 | 功能 |
|--------|------|
| <kbd>e</kbd> | 打开编辑器（需启用） |
| <kbd>Ctrl</kbd>/<kbd>Cmd</kbd> + <kbd>S</kbd> | 保存（编辑器内） |
| <kbd>Esc</kbd> | 关闭编辑器 |

## 实时协作 (Live)

| 快捷键 | 功能 |
|--------|------|
| <kbd>l</kbd> | 主控 ⇄ 被控切换（即使当前为关闭也直接进入循环） |
| <kbd>Shift</kbd> + <kbd>l</kbd> | 关闭 ⇄ 上一次激活模式切换 |

## AI 对话

| 快捷键 | 功能 |
|--------|------|
| <kbd>c</kbd> | 按默认形式打开 AI 对话（页内面板或独立窗口） |
| <kbd>Shift</kbd> + <kbd>c</kbd> | 按相反形式打开（临时反转一次默认设置） |

按 <kbd>Shift</kbd> 点击聊天小球或正文选区上的 **聊聊** 按钮也遵循相同的反转逻辑。默认形式在 **设置 → AI 对话 → 默认的对话框形式** 中选择。

## 平台差异

Markon 会自动检测 OS：

- **macOS** — 使用 <kbd>Cmd</kbd>（⌘）
- **Windows / Linux** — 使用 <kbd>Ctrl</kbd>

快捷键面板里会根据当前平台显示对应图标。

## 自定义快捷键

**全局设置 → 快捷键** 可自定义绑定：点击任意快捷键右侧的输入区，按下新组合键即可。

- **恢复默认** — 一键重置所有快捷键到出厂设置
- 自定义会持久化到配置文件，所有工作区通用
